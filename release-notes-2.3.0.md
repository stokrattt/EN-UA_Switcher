### 🚀 Що нового (Release 2.3.0)

* **Переписано обробку адресного рядка Chrome/Edge:** коли omnibox надає writable ValuePattern через UI Automation, корекція відбувається тихо через прямий виклик UIA — без Ctrl+A, мигання виділеного тексту та маніпуляцій з буфером обміну.
* **Clipboard-based фолбек для нестандартних випадків:** якщо ValuePattern недоступний, використовується старий повний rewrite (Ctrl+A → копіювання → токенізація → вставка).
* **Жорсткіша детекція Chrome garbage-scan:** коли ≥3 послідовних scan-кодів виявляється в буфері, перевага надається VK-інтерпретації — зменшує хибні конверсії в Electron-подібних вікнах.
* **Додано евристики для коротких українських слів із закінченням `-х`** та захист коротких технічних акронімів від помилкової EN→UA конверсії.
* **Розширене покриття регресійних тестів** за реальними сценаріями з user-reported проблем.
